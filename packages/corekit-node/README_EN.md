# @qlover/corekit-node

Node.js environment core tools for the fe-base toolkit.

## Installation

```bash
npm install @qlover/corekit-node
# or
pnpm add @qlover/corekit-node
```

## Usage

```typescript
import { /* utilities */ } from '@qlover/corekit-node';

// Use the utilities in your Node.js application
```

## Features

- Node.js specific utilities
- File system operations
- Process management
- And more...
