# @qlover/code2markdown

A tool to convert code to markdown documentation.

## Installation

```bash
npm install @qlover/code2markdown
# or
pnpm add @qlover/code2markdown
```

## Usage

```bash
npx fe-code2md -p example/example.ts
```

## Features

- Convert code files to markdown format
- Support multiple programming languages
- Generate well-formatted documentation
